# CIMIC Project Governance Framework

## Cost Control Thresholds

| Metric | Green | Amber | Red |
|--------|-------|-------|-----|
| Cost Variance (CV) | > -5% | -5% to -10% | < -10% |
| Schedule Performance Index (SPI) | > 0.95 | 0.85 - 0.95 | < 0.85 |
| Cost Performance Index (CPI) | > 0.95 | 0.85 - 0.95 | < 0.85 |
| Earned Value (EV) vs Plan | > 95% | 85-95% | < 85% |

## Approval Authority

| Change Value (AUD) | Approver |
|--------------------|----------|
| < $500K | Project Manager |
| $500K – $2M | Senior Project Manager |
| $2M – $5M | Divisional Director |
| $5M – $20M | Executive Committee |
| > $20M | Board of Directors |

## Reporting Cadence

- **Daily:** Site diary, safety observations
- **Weekly:** Project status dashboard, progress photos
- **Fortnightly:** Cost forecast update
- **Monthly:** Financial performance report, risk register update
- **Quarterly:** Board summary with strategic risk assessment
- **Annually:** Project audit and lessons learned

## Earned Value Management (EVM) Standards

All projects with contract value > AUD 50M must use EVM methodology:

- **Budget at Completion (BAC):** Total approved budget
- **Earned Value (EV):** Value of work completed to date
- **Actual Cost (AC):** Total costs incurred to date
- **Planned Value (PV):** Budgeted cost at this point in time
- **Cost Variance (CV):** EV - AC
- **Schedule Variance (SV):** EV - PV
- **Cost Performance Index (CPI):** EV / AC
- **Schedule Performance Index (SPI):** EV / PV
- **Estimate at Completion (EAC):** BAC / CPI

## Risk Categories

| Category | Examples |
|----------|---------|
| Technical | Design changes, unforeseen ground conditions |
| Commercial | Client variations, contract disputes, supply chain disruption |
| Environmental | Weather delays, environmental compliance issues |
| Safety | Incident escalation, regulatory changes |
| Political | Approval delays, policy changes, community opposition |
| Financial | Currency fluctuation, material price increases, funding delays |
